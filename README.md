## Quick Start

If you don't know to much about vim or nvim just follow the next steps:

```
$ git clone https://github.com/AndresMpa/My-NVIM-configuration.git
$ mv /My-NVIM-configuration nvim
$ mv /nvim ~/.config
$ nvim
```
